# Changelog

All notable changes to this project are documented here. The format follows
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/) and the project follows
[Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2026-06-04

### Added

- Themes: Monokai Pro, Monokai Pro Ristretto, Monokai Pro Octagon, Vira Teal, Vira Ocean (17 → 22).
- Vira file-icon pack, selectable next to Material and Minimal.
- Options live preview renders the selected icon pack.

### Changed

- Build derives the version from the latest git tag; `package.json` is only a fallback.
- README: replaced the badge grid with star-history and commit-activity charts; trimmed copy.

## [0.1.0] - 2026-06-04

### Added

- Initial release. File-tree sidebar for GitHub: lazy-loaded folders, fuzzy search, PR panel,
  single-file and ZIP downloads, bookmarks, repo/folder/file sizes, 17 themes, Material icons,
  8 fonts, dockable/resizable/pinnable layout, and GitHub Enterprise opt-in.
